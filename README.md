# Landing Page - The Odin Project

Este projeto é uma implementação de uma página de destino completa, desenvolvida como parte do currículo de Fundamentos do **The Odin Project**.

## Objetivo
Construir uma interface fiel ao design proposto, praticando a estruturação de layouts com **HTML5** e o uso avançado de **CSS Flexbox**.

## Tecnologias
- **HTML5**: Estrutura semântica das seções.
- **CSS3**: Posicionamento com Flexbox, estilização de botões, bordas arredondadas e tipografia.
- **Google Fonts**: Família de fontes Roboto (pesos 300 a 900).

## O que foi praticado
- Alinhamento de elementos em colunas e linhas.
- Uso de `justify-content` e `align-items` para centralização.
- Criação de componentes como botões e cards (info boxes).
- Importação correta de pesos de fontes externas para garantir a fidelidade visual.
